# weatherzone

**Archived.** Ruby client for the Weatherzone web service, last actively developed in 2012. Dependencies (`nokogiri ~> 1.5`, `activesupport ~> 3.1`) are incompatible with modern Ruby.

Special thanks to [@amasses](https://github.com/amasses) for [PR #1](https://github.com/benaskins/weatherzone/pull/1) — merged 15 years late, but merged.
